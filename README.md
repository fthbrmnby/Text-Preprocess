# Text-Preprocess
Text preprocessing pipeline for my [graduation](https://github.com/fthbrmnby/Graduation_Project) project. Pipeline includes sentence boundary detection, sentence tokenizer, stemmer, disambugiator and POS TAG. This pipeline uses Turkish NLP library [zemberek-nlp](https://github.com/ahmetaa/zemberek-nlp) by Ahmet A. AkÄ±n and [Turkish Deasciifier for Java](https://github.com/ahmetb/turkish-deasciifier-java) by Ahmet Alp Balkan.

### Dataset
| Type | Number of Reviews |
| ---- |:-----------------:|
| Positive | 220,284 |
| Negative | 14,881 |

### Requirements
+ JAVA 8
+ Maven
